Almost all elements on an HTML page are just pieces of content wrapped in opening and closing HTML tags.
# Opening tag  
At the start of an HTML element
Enclosed in angle brackets <>

EG <p>

# Closing Tag
Closes the element 
Eg </p> 

List of predefined tags
